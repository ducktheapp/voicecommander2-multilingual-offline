# VoiceCommander 2.0 Multilingual Offline

Offline voice command system for Windows.  
No cloud. No data sent. Everything runs locally.

## 🚀 Project Website
https://voicecommander2multilingual.sourceforge.io/

## 📥 Direct Download
https://sourceforge.net/projects/voicecommander2multilingual/files/VoiceCommander2.zip/download

## 🧠 Features
- Voice control (Whisper)
- Multilingual (ITA / ENG / FRA / ESP / DEU)
- Open/close apps
- Voice to text
- Fully offline
